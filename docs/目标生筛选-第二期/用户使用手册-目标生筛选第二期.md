# 目标生筛选第二期 用户使用手册

- 文档版本: v1
- 更新时间: 2026-04-06
- 适用对象: 班主任、年级组、教务老师

## 1. 功能概览

目标生筛选第二期包含三块核心能力:
- 高级筛选: 按多条件组合筛选目标生。
- 规则管理: 保存、复用、编辑、删除筛选规则。
- 变化追踪: 把筛选结果保存为快照，做跨考试对比，导出报告。

## 2. 使用前准备

- 账号已登录。
- 具备写权限角色（admin / grade_manager / staff）可执行筛选、保存快照、对比、规则写操作。
- 已存在考试与成绩数据。

## 3. 高级筛选（入口: 目标生高级筛选）

### 3.1 选择年级和考试
1. 先选年级。
2. 再选对应考试。
3. 若未选择年级，考试下拉不可用。

### 3.2 配置筛选条件
1. 在条件构建区新增条件。
2. 每条条件需选择:
   - 科目（含总分）
   - 维度（年级/班级）
   - 运算符（前 N / 后 N / 名次区间）
3. 系统支持 AND/OR 逻辑组合。

### 3.3 启动筛选并查看结果
1. 点击开始筛选。
2. 结果页展示:
   - 目标生总人数
   - 条件对应分数与排名列
   - 结果表导出 Excel

## 4. 规则管理（入口: 我的规则）

### 4.1 新建规则
1. 点击新建规则。
2. 填写规则名称。
3. 配置逻辑与条件。
4. 保存后可在高级筛选页一键回填。

### 4.2 编辑规则
1. 在规则列表点击编辑。
2. 修改名称或条件。
3. 保存后立即生效。

### 4.3 删除规则
1. 在规则列表点击删除。
2. 二次确认后删除。
3. 删除成功会返回“规则删除成功”。

## 5. 变化追踪（入口: 变化追踪）

### 5.1 保存快照
1. 在高级筛选结果页点击保存为快照。
2. 输入快照名称并确认。
3. 可选择保存后直接跳转变化追踪页。

### 5.2 选择快照并对比
1. 在历史快照列表中分别设置:
   - 基准快照
   - 对比快照
2. 点击开始对比分析。
3. 系统展示:
   - 新增名单
   - 退出名单
   - 保留名单
   - 排名变化

### 5.3 导出对比结果
- 每个子表可单独导出 Excel。
- 支持“全部导出”，一个文件内含三个 sheet（新增/退出/保留）。

### 5.4 删除快照
1. 在快照列表操作列点击删除图标。
2. 删除后列表自动刷新。
3. 如删除导致分页变化，页面会自动回到有效页。

## 6. 推荐操作流程

1. 在高级筛选页设置本次分析条件。
2. 查看结果并保存快照（例如月考、期中、期末各保存一次）。
3. 进入变化追踪，选择相邻阶段快照做对比。
4. 导出结果给年级组复核与跟进。

## 7. 常见问题

### 7.1 筛选失败
- 检查是否已选择考试。
- 检查条件值是否合法（如 top_n 必须为正整数）。

### 7.2 保存快照失败
- 快照名称不能为空。
- result_snapshot.count 必须与 student_ids 数量一致（系统通常自动保证）。

### 7.3 看不到他人的规则或快照
- 规则和快照是按用户隔离的，只能看到自己的数据。

### 7.4 无法执行写操作
- 账号角色可能是 subject_teacher，写接口会返回 403。
- 请联系管理员分配 staff / grade_manager / admin 角色。

## 8. 验收清单（业务侧）

- 可完成至少 1 次高级筛选并看到结果。
- 可保存 2 份快照并完成 1 次对比。
- 可成功导出单表和全部对比结果。
- 可在规则页完成新建、编辑、删除任一操作。
