## 1. 目的

构建可重复的验收基线，确保“目标生筛选”结果可由人工复核并与系统输出逐条对比。

---

## 2. 样本准备标准

1. 覆盖年级：至少2个（建议初二、初三）。
2. 覆盖考试：每个年级至少5场。
3. 覆盖缺考：至少包含10名有缺考记录学生。
4. 覆盖并列排名：至少包含1场并列排名样例。
5. 数据来源：以生产脱敏副本或稳定测试库为准。
