白云实验学校管理系统使用教程（面向日常使用者）

版本：2026年3月  
适用对象：教师、年级组、教务相关使用人员  
说明：本教程只讲“怎么用”，不涉及系统维护与技术内容。你可直接复制到 Word，并在“截图位置”处插入图片。

一、开始使用

1. 登录系统  
（1）打开系统登录页。  
（2）输入账号和密码。  
（3）点击登录，进入系统首页。  
【截图位置1：登录页面】  
【截图位置2：登录成功后首页】

2. 主菜单说明  
登录后，左侧可看到以下常用功能：  
学生信息、考试管理、成绩管理、成绩查询、成绩分析。  
【截图位置3：左侧菜单总览】

二、学生管理

1. 学生列表查看  
进入“学生信息”后，可看到学生总数、在读、已毕业、休学等统计。  
下方是学生列表，可查看学号、姓名、性别、班级、年级、状态。  
【截图位置4：学生列表与统计卡片】

2. 学生筛选与搜索  
可按以下条件筛选：  
学号/姓名关键字、年级、班级、状态。  
输入或选择后，列表会更新。点击“重置过滤条件”可恢复默认。  
【截图位置5：筛选区域】

3. 新增学生  
（1）点击“新增学生”。  
（2）填写信息（学号、姓名、年级、班级为重点项）。  
（3）点击“保存记录”。  
（4）保存成功后会自动返回学生列表。  
【截图位置6：新增学生页面】

4. 编辑学生信息  
（1）在学生列表中找到目标学生。  
（2）点击该行“编辑”按钮。  
（3）修改后点击“保存修改”。  
【截图位置7：编辑学生页面】

5. 删除单个学生  
（1）在学生列表中点击“删除”。  
（2）系统会弹出确认提示。  
（3）确认后删除。  
提示：删除后不可恢复，请谨慎操作。  
【截图位置8：删除确认提示】

6. 单个学生状态调整  
在学生列表每一行可直接修改状态（如在读、休学、毕业等），选择后确认即可生效。  
【截图位置9：状态切换操作】

7. 批量操作（重点）  
先勾选学生，再进行以下操作：  
（1）批量修改状态。  
（2）批量删除。  
（3）批量升年级。  
（4）批量毕业。  
【截图位置10：批量操作区域】

8. 批量升年级  
（1）在学生列表勾选学生，点击“批量升年级”。  
（2）选择目标年级，可选“限定当前年级”。  
（3）可按需勾选“自动创建缺失班级”。  
（4）点击“确认升年级”。  
【截图位置11：批量升年级页面】

9. 批量导入学生  
（1）点击“批量导入”。  
（2）先下载模板。  
（3）按模板填写后上传 Excel。  
（4）点击“开始导入”，查看成功/失败结果。  
（5）点击“完成并刷新数据”。  
【截图位置12：批量导入弹窗】  
【截图位置13：导入结果页面】

三、考试管理

1. 考试列表查看与筛选  
进入“考试管理”，可按学年、年级筛选考试，支持分页查看。  
【截图位置14：考试列表页面】

2. 创建考试（两步）  
第一步：填写基本信息  
学年、考试名称、考试日期、适用年级、说明。  
第二步：配置科目  
系统会给出常用科目，可调整科目并填写每科满分。  
完成后提交保存。  
【截图位置15：创建考试-基本信息】  
【截图位置16：创建考试-科目配置】

3. 编辑考试  
在考试列表点击“编辑”，进入后同样分两步修改：  
先改基本信息，再改科目和满分，保存后返回列表。  
【截图位置17：编辑考试页面】

4. 删除考试  
在考试列表点击“删除”，确认后执行。  
提示：删除考试会同时移除该考试对应成绩，请先确认。  
【截图位置18：删除考试确认弹窗】

四、成绩管理（录入、编辑、批量处理）

1. 成绩总览与筛选  
进入“成绩管理”后，可按学号、姓名、考试、年级、班级、科目筛选。  
支持分页与每页条数切换。  
【截图位置19：成绩管理主页面】

2. 手动新增成绩  
（1）点击“手动新增成绩”。  
（2）搜索并选择学生。  
（3）选择考试。  
（4）录入各科成绩（可留空未录入科目）。  
（5）点击“保存成绩”。  
若提示重复录入，可直接跳转到“编辑成绩”。  
【截图位置20：手动新增成绩页面】

3. 编辑已有成绩  
（1）在成绩列表中点击“编辑成绩”。  
（2）修改各科分数后保存。  
（3）保存成功后返回成绩列表。  
【截图位置21：编辑成绩页面】

4. 批量导入成绩  
（1）点击“批量导入”。  
（2）先选择考试，再选择 Excel 文件。  
（3）可先下载模板，按模板填写后上传。  
（4）查看导入结果（成功数、失败数、失败原因）。  
【截图位置22：批量导入成绩弹窗】  
【截图位置23：成绩导入结果】

5. 批量删除成绩  
（1）可先筛选，再全选。  
（2）支持“删除选中项”或“删除筛选结果”。  
（3）系统会二次确认，确认后执行。  
【截图位置24：批量删除操作区】

可进行更细致的查询，不会直接修改数据。

【截图位置26：查询条件区】

可点击列头进行升序/降序查看（如某科、总分、级排）。

点击“导出Excel”，可下载当前筛选条件下的数据。  
【截图位置28：导出按钮与下载结果示意】

六、成绩分析（班级/年级分析 + 个人分析）

A. 班级/年级分析

1. 分析入口选择  
进入“成绩分析-班级/年级分析”，依次选择：  
学年 → 考试 → 年级 → 班级。  
然后点击“开始分析”。

2. 三种分析模式  
（1）选择1个班级：进入单班级分析。  
（2）选择多个班级：进入多班对比分析。  
（3）选择“所有班级”：进入年级分析。  

3. 单班级分析可看内容  
班级总体情况、科目表现、分数分布、学生排名等。  
【截图位置30：单班级分析页】

4. 多班对比分析可看内容  
多个班级在各科平均分、整体表现、分布对比等。  
【截图位置31：多班对比分析页】

5. 年级分析可看内容  
年级总体水平、班级对比、科目对比、分段分布等。  
【截图位置32：年级分析页】

B. 个人成绩分析

年级 → 班级 → 学生，然后点击“开始分析”。

（3）单次考试各科成绩对比。

七、常见问题处理（给使用者）

1. 选不到数据  
先检查是否已选上一级条件（如先选年级再选班级）。  
若仍无数据，可能该条件下暂时没有记录。

2. 导入失败  
先下载模板，按模板填写后再上传。  
确保学号、姓名、科目名称填写正确。

3. 操作后没看到变化  
可尝试重新筛选、翻页后返回，或刷新当前页面。

4. 删除前提醒  
删除类操作通常不可恢复，建议先筛选确认再执行。
