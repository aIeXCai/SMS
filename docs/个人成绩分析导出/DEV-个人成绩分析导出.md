## 6. 关键实现细则（防返工）
### 6.1 口径与公式
1. 较上次总分变化 = 本次总分 - 上次总分
2. 较上次年级排名变化 = 上次年级排名 - 本次年级排名（正数=提升）
3. 相对首场排名变化 = 首场年级排名 - 本次年级排名（正数=提升）
4. 科目排名变化 = 上次科目级排 - 本次科目级排（正数=提升）
5. 若不可计算，统一填 `-`

### 6.2 排序与缺失值
1. 考试按日期升序
2. 科目列按 `SUBJECT_CHOICES` 顺序
3. 不在 `SUBJECT_CHOICES` 的科目追加末尾
4. 缺考、缺排名、缺科目统一填 `-`

### 6.3 样式最小标准
1. 每个工作表有清晰表头
2. 排名变化列应用条件格式
3. 数字列右对齐，文本列左对齐
4. 顶部冻结（有长表时）提升可读性

### 6.4 图表最小标准（新增）
1. 本期必须包含 4 张图表（总分趋势、年级排名趋势、滚动均分、各科级排变化）
2. 图表数据源必须来自可见表格区域，禁止引用隐藏临时区
3. 排名趋势图 Y 轴反向，保证“名次提升”视觉向上
4. 图表失败不影响导出，必须写入降级提示文本
